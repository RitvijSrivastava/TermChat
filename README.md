# TERM CHAT

Terminal based chat. UI built (soon) using ncurses. Built on python using sockets and threading

## How to run?

Change the <i>server</i> and <i>port</i> according to your ipv4 address


FOR SERVER:

``` 
    python3 server.py
```

FOR CLIENT:

```
    python3 client.py
```
